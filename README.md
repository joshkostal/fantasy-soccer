## Stack

* Client
  * Ionic
  * React
  * Apollo
  * TypeScript
* Server
  * GraphQL
  * TypeScript
  * graphql-yoga
  * Prisma
* Database
  * PostgresSql

## Useful commands

* Prisma
  * npx prisma migrate reset
    * Republishes database with seed data

  * npx prisma migrate dev --name {migration-name}
    * Create new migration with name

* Server
  * npm start

* Client
  * npm start
